## Project Goal

Moon Lamp Version 2 was inspired by the first downloaded moon lamp build. This
version pushed the idea further by turning the lamp into a rotating lithophane
moon with RGB lighting and Wi-Fi control.

The goal was to make the lamp feel less like a static object and more like an
interactive lighting piece.

## What I Built

- Rotating moon lamp concept
- Lithophane-style moon surface for light-driven detail
- RGB lighting system
- Wi-Fi-enabled control approach
- Mechanical design considerations for smooth rotation
- Electronics layout for lighting, motion, and control

## Development Notes

Version 2 grew directly from the questions raised by Version 1. Instead of only
placing a light inside a printed moon, I wanted to control the experience:
colour, brightness, movement, and interaction.

Adding rotation changed the design problem. The lamp now needed mechanical
stability, clean wiring, and careful placement of the light source so the moon
could move without ruining the visual effect.

## Results

The second version became a more original and technically ambitious lamp. It
combined 3D printing, lighting design, motion, and connected control into one
project.

## What I Learned

- How to turn a reference build into a more personal design
- How RGB lighting changes the mood and readability of a lithophane
- Why moving decorative objects need mechanical and electrical planning together
- How Wi-Fi control can make a physical object feel more interactive
