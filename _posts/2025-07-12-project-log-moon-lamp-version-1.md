## Project Goal

Moon Lamp Version 1 started from a model downloaded from the internet. The goal
was to print and assemble a moon lamp, understand how the surface detail worked,
and learn what made the object visually convincing when lit from the inside.

This version was not about claiming the original model as my own design. It was
about learning from an existing object by making it, observing it, and finding
ways it could become something more personal.

## What I Built

- 3D printed moon lamp shell from an online model
- Internal lighting setup
- Basic assembly for a glowing moon effect
- First tests of light diffusion through printed material
- Notes for improving the idea in a future version

## Development Notes

The most useful part of Version 1 was seeing how the print behaved with light.
Wall thickness, filament color, surface texture, and LED placement all changed
the final look.

Building the downloaded version helped me understand what I liked about the
object and what I wanted to change. That process later inspired Version 2.

## Results

The lamp worked as a first moon light and became a reference point for the next
iteration. It showed that the idea had potential, but also that a more original
version could add motion, better lighting control, and a stronger interaction
concept.

## What I Learned

- How internal lighting reveals 3D print details
- Why diffusion and wall thickness matter in lamp design
- How an existing model can be used as a learning prototype
- What features would make the lamp feel more complete

## Next Iteration

- Design a more original version
- Add rotation for a more dynamic display
- Explore lithophane-style moon detail
- Add RGB lighting and wireless control
